# propeller-graduation-cap

Materials
-------------------------------------------------------------
- DC Motor (works best with those circular coreless ones)
- Battery pack
- Batteries
- Single-Pole switch
- Soldering iron/DuPont wires
- Construction Paper
- Glue/Tape/Velcro
- 3D Printer (or cardboard/wood if you want to get even more DIY)
